lingvo.tools.audio_lib.ExtractLogMelFeatures(wav_bytes_t)[source]¶

Create Log-Mel Filterbank Features from raw bytes.

Parameters

wav_bytes_t – Tensor representing raw wav file as a string of bytes. It is currently assumed that the wav file is encoded at 16KHz (see DecodeWav, below).

Returns

A Tensor representing three stacked log-Mel filterbank energies, sub-sampled every three frames.
